# 以太坊设计与实现

这是一本持续写作中的电子书，欢迎关注。

本电子书是对以太坊区块链的实现原理和程序实现进行详解，力图展示区块链2.0先锋以太坊智能合约编程平台设计思想与代码实现。

电子书的目录为本书内容结构，如果可以点击查阅，说明我已写完这一小节。
